# pip: command not found

执行下面命令进行脚本下载

```
curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py
```

![img](https://img-blog.csdnimg.cn/20201224150056763.png)

运行安装脚本

```
sudo python get-pip.py
```
